O que é API VIA CEP?

Usada para consultar o CEP, ou seja, os dados fornecidos com ele.

Site oficial do VIA CEP: clique aqui


Como funciona?

JSON: https://viacep.com.br/ws/{cep}/json/

Digitando o CEP com os 8 dígitos sem caracteres especiais. Exemplo: 01001000

No link do json escreva assim: https://viacep.com.br/ws/01001000/json/
